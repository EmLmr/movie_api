Movie API Documentation

Objectives

This API will enable users to receive information on movies, directors and genres.
Users will be able to create a profile in order to save a list of their favorite movies.
The user can also log in to their profile to enter and modify personal data.

Essential features in detail

  • Return a list of ALL movies to the user
  • Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name/title (e.g., “Thriller”)
  • Return data about a director (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update their user info (username, password, email, date of birth)
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites
  • Allow existing users to deregister

Return ALL users. /users GET None A JSON object holding data about all the users in the database.
Return data about a user, by username. /users/:Username GET None A JSON object holding data about the user:
{
"Username": "JohnDoe",
"Password": "myPassword123!",
"Email": "johnDoe@gmail.com",
"Birthday": "2000-06-21"

}
Allow new users to register. /users POST A JSON object receiving the required data to register the user: username, password, email, date of birth.
{
"Username": "JohnDoe",
"Password": "myPassword123!",
"Email": "johnDoe@gmail.com",
"Birthday": "2000-06-21"
} 		A JSON object holding data about the user that was registered, creating a unique user ID.
{
"_id": "611131dca634d43cf881dfb1",
"Username": "JohnDoe",
"Password": "myPassword123!",
"Email": "johnDoe@gmail.com",
"Birthday": "2000-06-21"
}

If the user already exists, user will see the following message:
"[Username] already exists. Please choose another username."
Allow users to update their user info. /users/:Username PUT A JSON object with the required fields to update: username, password, email, date of birth.
{
"Username": "JohnDoe2",
"Password": "myNewPassword123!",
"Email": "johnDoe2@gmail.com",
"Birthday": "2000-06-12",
}
A JSON object holding the updated user data, along with the unchanged user ID, and unchanged favorite movies list.
{
"_id": "611131dca634d43cf881dfb1",
"Username": "JohnDoe2",
"Password": "myNewPassword123!",
"Email": "johnDoe2@gmail.com",
"Birthday": "2000-06-12",
"FavoriteMovies": []
}
Allow existing users to deregister, by username. /users/:Username DELETE None A plain text message indicating whether deregistration was successfull or not.
Allow users to add a movie to their list of favorites, by movie ID. /users/:Username/movies/:MovieID POST None A JSON object holding the user data including the updated list of favorite movies. {
"_id": "611131dca634d43cf881dfb1",
"Username": "JohnDoe2",
"Password": "myNewPassword123!",
"Email": "johnDoe2@gmail.com",
"Birthday": "2000-06-12",
"FavoriteMovies: [movie1, movie2, newMovie]"
}
Allow users to remove a movie from their list of favorites. /users/:Username/movies/:MovieID DELETE None A JSON object holding data about the user's updated favorite movies along with all user data:
{
"_id": "611131dca634d43cf881dfb1",
"Username": "JohnDoe2",
"Password": "myNewPassword123!",
"Email": "johnDoe2@gmail.com",
"Birthday": "2000-06-12",
